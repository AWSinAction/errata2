## Using this project

[Install Hugo](http://gohugo.io/overview/installing/) to build the site.

### Run locally

```
make run
```

Open http://localhost:1313/ to see the site.

### Generate HTML

```
make build
```

send file `./public/index.html` to Susan Harkins <suha@manning.com>
